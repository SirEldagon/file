generateFile создает файл заданного размера, заполняя его случайными числами.
sortFile (пока что заглушка) должна содержать логику external merge sort. Этот алгоритм разбивает файл на части, сортирует их по отдельности, а затем сливает в отсортированный файл.
В main задаются имя файла, размер и размер чанка для сортировки.
